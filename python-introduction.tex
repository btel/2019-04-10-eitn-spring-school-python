    \hypertarget{what-is-numpy}{%
\subsection{What is numpy ?}\label{what-is-numpy}}

    \begin{itemize}
\item
  memory-efficient container for multi-dimensional homogeneous (mainly
  numerical) data (NumPy array)
\item
  fast vectorised operations on arrays
\item
  library general purpose functions: data reading/writing, linear
  algebra, FFT etc. (more scientific functions are implemented in SciPy
  lecture)
\item
  main applications: signal processing, image processing, analysis of
  raw data from measurment instruments
\item
  coherent interface across the entire Python data ecosystem
\end{itemize}

    \textbf{Rules of broadcasting}

Two numpy arrays can broadcast when their dimensions are compatible. A
dimension of the two arrays is compatible when:

\begin{itemize}
\tightlist
\item
  they are equal, or
\item
  one of them is 1
\end{itemize}
